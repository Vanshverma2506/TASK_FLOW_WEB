# TaskFlow 🚀

A simple full-stack task manager with role-based access (Admin / Member).

---

## Features

- Login / Register (JWT auth)
- Admin:
  - Create projects
  - Assign tasks
- Member:
  - View & update tasks
- Dashboard (tasks, pending, completed, overdue)
- Clean UI with toast notifications

---

## Tech Stack

- React + Tailwind
- Node.js + Express
- MongoDB
- JWT

---

## Run Locally

```bash
git clone <repo-link>
cd taskflow
```

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```
